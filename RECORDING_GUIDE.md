# 武亮快手直播录制指南

## 🔍 观看入口

打开快手APP，搜索以下任一关键词：
- "峰学蔚来武亮"
- "峰学未来"
- 武亮快手ID（在快手搜索用户）

> ⚠️ 武亮已于 **2026年6月15日** 从抖音整体迁移到快手。
> 快手支持长直播回放，连麦平均47分钟不划走。

## 📹 录制方案

### 方案A：手机自带录屏（最简单）
1. 打开快手APP → 找到武亮直播回放
2. 下拉控制中心 → 点击"屏幕录制"
3. 播放完整回放 → 录制结束保存

### 方案B：OBS Studio（电脑端，推荐）
1. 下载 OBS Studio (https://obsproject.com)
2. 添加"窗口捕获" → 选择快手PC版/模拟器窗口
3. 添加"音频输出捕获" → 录制系统声音
4. 开始录制 → 播放直播回放

### 方案C：快抖直播录制助手（自动化）
1. 下载"快抖直播录制助手"
2. 添加武亮快手直播间链接
3. 设置开播自动录制 + 断线续录

## 📝 录制目标

| 优先级 | 内容类型 | 目标数量 |
|--------|----------|---------|
| P0 | 完整连麦对话（家长提问→武亮回答） | 5-10 组 |
| P1 | 分省志愿填报策略 | 3-5 场 |
| P2 | 专业深度分析（医学/电气/计算机/法学） | 3-5 个 |

## 🔄 录制后处理

1. 用 ffmpeg 提取音频: `ffmpeg -i 录制文件.mp4 -vn 音频.wav`
2. 用 Whisper 转录: `whisper 音频.wav --language zh --model tiny`
3. 保存 `.txt` 到项目的 `transcripts/` 目录
4. 告诉我文件名，我会自动更新 Skill
