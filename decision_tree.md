## 场景1: 物理类考生志愿规划 (主决策流)

```mermaid
flowchart TD
    START["👤 家长/考生提问"]
    
    START --> Q1{"① 分数段确定 [ref:S005]"}
    
    Q1 -- "特殊类型线以上(600+)" --> Q2{"② 家庭背景判断 [ref:S005][GAP:阈值待验证]"}
    Q1 -- "本科线-特殊类型线" --> Q3{"② 家庭背景判断"}
    Q1 -- "本科线以下" --> LOW["进入低分段专项流程 → 场景4"]
    
    Q2 -- "普通家庭(就业优先)" --> PATH_A["路径A: 理工打底+医学备选 [ref:S005]"]
    Q2 -- "家庭资源充足" --> PATH_B["路径B: 地域优先+专业为王 [ref:S005]"]
    
    PATH_A --> A1{"就业取向? [ref:S005]"}
    A1 -- "求稳/体制内" --> A2["推荐: 临床医学5+3一体化<br/>电气工程(国网路径)<br/>师范类定向就业 [ref:S005]"]
    A1 -- "求高薪/市场化" --> A3["推荐: 计算机/电子信息类<br/>注意: 需选城市好的院校 [GAP:城市权重待验]"]
    
    PATH_B --> B1{"地域偏好? [ref:S005]"}
    B1 -- "留本省" --> B2["锁定省会城市院校<br/>如河北→河北医大临床 [ref:S005]"]
    B1 -- "可出省" --> B3["专业实力排名优先<br/>【GAP:具体院校推荐阈值缺失】"]
    
    Q3 -- "普通家庭" --> Q4{"分数接近本科线? [GAP:压线生案例缺失]"}
    Q4 -- "是" --> Q5{"专业适配度?"}
    Q4 -- "否(安全区间)" --> PATH_A
    
    Q5 -- "有明确专业方向" --> Q6["冲稳保设置: 冲≤20% | 稳≥50% | 保≥30% [ref:S005]"]
    Q5 -- "无明确方向" --> Q7["⚠️ 先确定专业方向再谈学校 [ref:S005]<br/>反对盲目冲高"]
    
    Q6 --> OUT["输出方案: 院校清单+专业排序"]
    
    style START fill:#e1f5fe
    style LOW fill:#fff3e0
    style Q7 fill:#ffebee
    style OUT fill:#c8e6c9
```

**溯源检查:**
- Q1→Q2 逻辑: [ref:S005] "物理类600分：理工打底、医学备选、地域优先、专业为王"
- PATH_A: [ref:S005] "就业优先、兼顾升学"
- A2 具体推荐: [ref:S005] "临床医学5+3一体化"
- Q7: [ref:S005] "坚决反对'为了学校读天坑专业'"
- **已知 GAP**: 分数段精确阈值、压线生具体策略、城市权重计算
