## 五、武亮择校三要素 [S036]

```
报本科(特别是到不了985的话)看三个点:
① 就业 — 哪些企业来校招？招本科生还是研究生？
② 推免 — 有没有推免资格？保研率多少？
③ 硕博点 — 有没有对应硕士点？导师数量？
→ 别管学科评估(跟本科没关系)
```

## 七、特殊招生路径 (艺考 / 强基 / 专项)

### 🎨 艺考路径
> ⚠️ 武亮对艺考的直接素材极少。仅有一条：对艺考提问回答"不太建议""也可以考虑"[S026]，被批评为"模糊摇摆"。

| 路径 | 态度 | 建议 | 确信度 |
|------|------|------|--------|
| 美术/设计类 | 有条件建议 | 城市经济>院校层次[S010]；设计类加班严重[峰学推断] | 中确信 |
| 音乐/表演类 | 谨慎 | 就业面极窄，头部效应明显 | [峰学推断] |
| 编导/传媒类 | 不推荐 | 传统媒体萎缩+AI冲击 | [峰学推断] |

**艺考决策条件**:
- IF 对设计有强烈兴趣 + 目标城市有设计产业 → 可考虑
- IF 纯艺术方向 + 家庭无法兜底 → 强烈不推荐
- IF 文化课>=本科线 + 艺考作为保底 → 警惕"为了上大学而艺考"

### 🎓 强基计划
> ⚠️ 武亮无直接观点。以下为峰学体系推断。

| 特征 | 说明 |
|------|------|
| 定位 | 国家基础学科拔尖人才培养(数/理/化/生/文/史/哲) |
| 适合人群 | 对基础学科有真实兴趣+家庭可支撑长线学术(硕博) |
| 不适合人群 | 普通家庭+求快速就业 |
| 优势 | 降分录取985、本硕博贯通培养、导师制 |
| 风险 | 专业锁定(不可转)、退出即失去优惠资格 |
| 峰学态度推断 | 适合"学术型"学生，普通家庭慎入(回报周期长) |
